# Amazon-Business-Performance-Dashboard
This project analyzes Amazon sales data to understand revenue, orders, customer behavior, and product performance  It helps in tracking overall business growth and identifying key revenue drivers

🔹 Tools Used

Excel

Python (Pandas)

Power BI

Power Query

DAX

🔹 Key Insights

Total Orders: 50K

Total Revenue: 32,867K

Total Quantity Sold: 150K

Total Reviews: 12,466K

Average Rating is around 3.0

Revenue is evenly distributed across all regions

Cash on Delivery and UPI generate higher revenue

Product categories like Beauty, Fashion, and Electronics perform well

🔹 Features

KPI Cards (Orders, Revenue, Quantity, Reviews)

Region-wise Sales Analysis

Payment Method Analysis

Product Category Revenue Breakdown

Monthly Revenue Trend

Customer Rating Analysis

Interactive Filters (Region, Category, Payment Method)

🔹 Project Workflow

Data cleaning and preprocessing using Excel and Python

Data transformation using Power Query

Data modeling in Power BI

DAX used for KPIs and calculations

Dashboard created for business insights

🔹 Business Impact

Helps identify top performing products

Supports better pricing and marketing strategies

Improves understanding of customer preferences

Helps in increasing overall revenue
